科学上网：VPS上搭建shadowsocks

2017年12月17日 93条评论 19,385次阅读 43人点赞

搭建shadowsocks梯子代理服务器翻墙实现科学上网，在VPS上搭建ss服务器实现科学上网，从零开始，搭建shadowsocks教程，搭建代理服务器，搭建shadowsocks代理实现科学上网。自己搭建ss做梯子翻墙实现科学上网，首先肯定要有一个在墙外的服务器作为跳板。一键安装shadowsocks&开启BBR加速脚本戳这里。Vultr KVM内核支持BBR加速与锐速加速，速度快，不限速服务器还可以用来作为个人网站的服务器。以Vultr作为搭建shadowsocks的教程示例。不要选搬瓦工包年！！IP被墙了换个IP要8刀！我选择的是Vultr，平均下来15块一个月（2.5刀），分给20个人用也是可以的，主要被墙了，用Snapshot方法换IP特别简单，只要一分钱~

这是一篇绝对适合新手的手把手搭建shadowsocks教程，耐心读完绝对可以成功~有任何问题欢迎留言，及时回复~

 

购买在GFW之外的代理服务器（图文教程：Vultr购买步骤）

  • vps注册

按照步骤注册就可以了：戳我戳我戳我注册~

  • 充值

选择左侧栏的Billing，选择个充值方式即可（支持支付宝Alipay，支持支付宝Alipay，支持支付宝Alipay）。目前2.5刀的服务器在Miami和New York地区有售（延迟相对其他地区稍高），其他都售罄了，赶紧抢购吧~

  • 配置VPS

选择左侧栏的Servers：Server Location是Los Angels和Tokyo的比较好（Vultr各节点官方测速地址）；Server Type选择Ubuntu 16.04（当然也可以不选择Ubuntu，本教程是基于Ubuntu 16.04的）,Server Size根据需求选择，Additional Features中勾选Enable IPv6。其他都可以默认，然后选择Deploy Now就可以了。

  • 查看相关信息

配置完服务器后，在Servers就可以看到自己服务的信息了，包括我们需要用到的IP和Password。得到IP和密码就可以用XShell进行连接了（Xshell连接Linux）。

 

在服务器上搭建shadowsocks

以下的步骤都是以root权限进行的（Vultr也默认是root用户），如果不是root，自行加sudo~

  • 更新软件源

1
apt-get update

  • 安装pip环境

1
apt-get install python-pip

  • 安装shadowsocks

1
pip install shadowsocks

此时，如果出现了提示版本太低，则按照提示更新

1
pip install --upgrade pip

如果提示没有setuptools模块，则安装setuptools

1
pip install setuptools

如果刚才shadowsocks安装成功则跳过这一步，某则继续安装shadowsocks

1
pip install shadowsocks

  • 编辑配置文件

1
vim /etc/shadowsocks.json

添加（vim编辑器基本使用，如果不知道怎么编辑，戳这里，如果只想看怎么处理这些配置文件，可以在公众号回复vim编辑器使用，会得到以这个配置文件为例的简单使用教程）：

1
2
3
4
5
6
7
8

{
    "server":"0.0.0.0",
    "server_port":8388,
    "local_address": "127.0.0.1",
    "local_port":1080,
    "password":"mypassword",
    "timeout":300,
    "method":"aes-256-cfb"
}

将其中的mypassword换成你想设置的密码（不需要跟你的VPS密码一致，这个密码你用来在客户端连接服务器ss的）。

  • 赋予文件权限

1
chmod 755 /etc/shadowsocks.json

  • 安装以支持这些加密方式

1
apt-get install python-m2crypto

  • 后台运行

1
ssserver -c /etc/shadowsocks.json -d start

  • 停止命令（这个命令是告诉你怎么停止shadowsocks的，不需要运行！不然就停止了你也就连不上了）

1
ssserver -c /etc/shadowsocks.json -d stop

  • 设置开机自启动

1
vim /etc/rc.local

exit 0

本机上搭建shadowsocks代理实现科学上网

各种客户端版本下载地址：各版本shadowsocks客户端下载地址

以Windows为例~（其他版本的客户端如果下载有问题，欢迎关注公众号flyzy小站，发送ss，会给你各种客户端的下载方法~）

在状态栏右击shadowsocks，勾选开机启动和启动系统代理，在系统代理模式中选择PAC模式，服务器->编辑服务器，用shadowsocks.json中配置的密码、加密方法以及你VPS的IP填好，保存即可~

PAC模式是指国内可以访问的站点直接访问，不能直接访问的再走shadowsocks代理~

OK！搭建shadowsocks完毕！翻墙吧，兄弟！Google
